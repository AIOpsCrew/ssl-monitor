About Domain Grouping

Use domain grouping to combine related domains that share the same SSL certificate:

  • No Grouping: Each domain is monitored separately.
  • Auto-detect Related Domains: Automatically groups www and non-www versions of the same domain (e.g., example.com and www.example.com).
  • Group by Domain Pattern: Domains with the same base domain are grouped (e.g., example.com, blog.example.com, store.example.com).
